# Birthday cEDH Gauntlet — Paquete de ocho mazos

Este paquete contiene ocho mazos completos de cEDH, de 100 cartas y sin límite de presupuesto, preparados para rotar entre jugadores.

## Roster

1. RogSi — Rograkh / Silas Renn
2. Blue Farm — Tymna / Kraum
3. Kinnan, Bonder Prodigy
4. Magda, Brazen Outlaw
5. Kefka, Court Mage
6. Ral, Monsoon Mage
7. Yuriko, the Tiger's Shadow
8. Etali, Primal Conqueror

## Formato

- Dos pods aleatorios de cuatro jugadores en cada ronda.
- Mazos asignados aleatoriamente.
- Tracking individual para intentar evitar repeticiones.
- Mazos entregados enfundados y con tokens, fichas, contadores y ayudas de memoria.
- Guías disponibles para consulta durante las partidas.

## Validación

- Las ocho listas contienen exactamente 100 cartas.
- Se comprobó singleton, salvo tierras básicas.
- Ral y Etali no requieren stickers.
