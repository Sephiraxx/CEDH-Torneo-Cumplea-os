# Birthday cEDH Gauntlet

Repositorio preparado para GitHub Pages con ocho listas de cEDH, guías de piloto en español e imports para Moxfield.

## Publicación

1. Subir el contenido de esta carpeta a la rama `main`.
2. Abrir **Settings → Pages**.
3. Elegir **Deploy from a branch**.
4. Seleccionar `main` y `/ (root)`.

La portada se genera desde `index.md`.
